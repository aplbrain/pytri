<h1 class="display-2">pytri</h1>

<br />

## Installation
<br />
Install `pytri` like any other Python package:

```shell
pip install pytri
```

## Usage
<br />
Import `pytri` in a Jupyter notebook:
```python
from pytri import pytri
```

Then invoke it in a new cell:
```python
p = pytri()
p.show()

p.axes()
```
