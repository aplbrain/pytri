## Installation and Configuration
- Clone the repository.
```shell
git clone https://github.com/iscoe/pytri.git
```
- Install all dependencies.
```shell
pip3 install -r requirements.txt
```

## Usage

```python
from pytri import pytri

p = pytri()

p.axes()
p.scatter([[1, 2, 3], [4, 5, 6]])
p.show()
```
